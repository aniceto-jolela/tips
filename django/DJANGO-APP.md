### Let's create a new `app`.
#
- Step 1
```shell
django-admin startapp blog
```
![1.png](../assets/django/app/1.png)
- Step 2 (blog)
### blog

- - ```shell 
    mkdir templates
    ```
- - ```shell
    mkdir blog
    ```
- - ```shell
    vim home.html
    ```
- - ```shell
    vim about.html
    ```
- - ```shell
    vim details.html
    ``` 

> [!NOTE]
> **mkdir** ⇢ Command to create dir.
> 
> **vim file_name** ⇢ Command to create or edit new file.
